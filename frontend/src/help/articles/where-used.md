# Where Used

Where Used helps Quality Tech and higher roles trace finished tanks from upstream material identifiers.

**Access:** Quality Tech (5.0) and above.

## How It Works

1. Enter one or more search values in **Heat**, **Coil**, and/or **Lot**.
2. Optionally filter by **Plant** (for users who can access multiple plants).
3. Click **Search**.
4. Review finished serial numbers that are connected through traceability logs.

## Returned Columns

| Column | Description |
|---|---|
| **Plant** | Plant name and code for the finished serial number. |
| **Serial Number** | Finished serial number. Click to open Serial Number Lookup. |
| **Production Number** | Product number for the finished tank. |
| **Tank Size** | Tank size in gallons. |
| **Date Completed (Hydro)** | Hydro completion timestamp. Uses Hydro inspection time first, then Hydro production-record time when inspection is missing. |
